Creates a process from an ELF file.

Reads an ELF file from a path and makes it into a running task, automatically being added to scheduling.

Parameters
pathThe path of the ELF file.
kernelWhether the process is EL1 (kernel) or EL0 (user).
pidThe process id of the task. Passing -1 sets the process id automatically (recommended).
argcThe arguments count to be passed.
argvThe arguments values to be passed.
envargcThe environment counts to be passed. Must be a multiple of 2.
envargvThe environment values to be passed. Must have a count multiple of 2.
Returns
Pointer to the process.
